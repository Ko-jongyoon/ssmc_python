[[파이썬 웹프로그래밍]]
1. 개발 환경 구축
 > 콘솔(터미널) 오픈
 > 확인 차원으로 현재 python 환경에 설치된 패키지 목록 확인
   pip list
 > 확인결과 flask가 없는것으로 확인되었으면 설치
   pip install flask
 > 설치 결과 확인
   pip list
 > 가상환경 virtualenv 환경에서 통상 설치 개발한다.
   이는 개발 환경(os측면)과 운용환경(AWS or docker)을 동일하게 맞추기 위한 절차
   당장 영향을 받는 부분이 없어서 일단 로컬에 설치 => 차후변경예정
